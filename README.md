# Hue Sort

A color sorting puzzle game. Swap tiles to reconstruct the hidden gradient.

## How to Play

- The four corner tiles are fixed — use them as your guide
- Tap a tile to select it, then tap another to swap
- Keep swapping until every tile matches the correct gradient

## Features

- 4×4 to 10×10 grids that scale with your level
- Undo support
- Random puzzle mode (Easy, Medium, Hard)
- Progress saved automatically
- No ads. No trackers. No data collection. Fully GDPR compliant.
